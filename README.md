## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/testimonial-grid-section.git
```

Go to the project directory

```bash
  cd testimonial-grid-section
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
